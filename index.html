🏥 진료비 내역서 변환기 Claude AI

병원 영수증·진료비 내역 사진(JPG/PNG) 또는 PDF를 업로드하면 Claude AI가 내용을 분석해 실손보험 청구용 엑셀 파일로 변환합니다. 양식이 병원마다 달라도 자동으로 항목을 인식합니다. 인식을 위해 이미지가 Anthropic(Claude) 서버로 전송됩니다.

1. Claude API 키

키는 이 브라우저에서 Anthropic API로만 전송되며 다른 서버로 가지 않습니다. 저장 시 localStorage에만 보관됩니다. 키 발급은 console.anthropic.com → API Keys에서 가능하며, 사용료는 키 소유자에게 부과됩니다(30페이지 기준 약 $0.2~0.3).

2. 파일 업로드

📄 여기를 클릭하거나 파일을 끌어다 놓으세요 (사진 JPG/PNG 또는 PDF, 여러 개 동시 선택 가능)

    페이지당 인식에 5~15초 정도 소요됩니다. 결과는 표에서 확인·수정한 뒤 다운로드하세요.

    3. 결과 확인 및 수정

    No.진료일자진료비 총액본인부담(급여)공단부담금전액본인부담비급여환자부담 총액비고

    노란 셀은 클릭하여 직접 수정할 수 있습니다. 진료일은 오름차순 정렬되며, 맨 아래 합계 행과 엑셀 합계(SUM 수식)는 자동 계산됩니다.
