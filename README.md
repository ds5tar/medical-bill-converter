# 진료비 내역서 변환기

병원 진료비 영수증·내역서 **사진(JPG/PNG) 또는 PDF**를 업로드하면 **여러 무료 AI가 자동 전환**되며 내용을 인식하여 실손의료보험 청구용 **엑셀 파일(.xlsx)** 로 변환해주는 웹 프로그램입니다. 가입·API 키·비용이 전혀 필요 없습니다.

## 무료 AI 자동 전환 (한도 걱정 없음)

하나의 AI가 사용량 한도에 걸리면 자동으로 다음 AI로 넘어갑니다.

1. **GPT-4o 비전** (Puter.js) — 구조화 인식, 가장 정확
2. **Mistral OCR** (Puter.js) — 무제한 OCR
3. **OCR.space** — Puter와 완전 독립된 무료 OCR
4. **Tesseract** — 브라우저 내장, 오프라인, 항상 동작 (최종 안전망)

마지막 Tesseract는 서버 없이 내 브라우저에서 처리하므로, 모든 온라인 AI가 막혀도 인식이 완전히 멈추지 않습니다.

**사용하기:** https://ds5tar.github.io/medical-bill-converter/

## 사용 방법

1. 진료비 영수증 사진 또는 PDF를 업로드합니다. (여러 개 동시 선택 가능)
2. "AI 인식 시작"을 누릅니다. 페이지당 10~30초 정도 걸립니다.
3. 인식 결과를 표에서 확인·수정합니다.
4. "엑셀 다운로드"를 누르면 완성된 파일이 저장됩니다.

## 알아둘 점

- 인식을 위해 이미지가 해당 AI 서버로 전송됩니다(마지막 Tesseract 단계는 전송 없이 브라우저에서 처리).
- Puter 계열 AI는 최초 1회 이용 동의 창이 뜨며(무료), 이후에는 뜨지 않습니다.
- 병원마다 양식이 달라도 AI가 항목을 자동 매핑하며, OCR 단계에서는 규칙 기반 파서가 항목을 채웁니다. 결과는 표에서 확인·수정하세요.

## 출력 엑셀 양식

시트 "진료비내역"에 환자·의료기관 정보와 진료일별 내역(진료일자, 진료비 총액, 본인부담금, 공단부담금, 전액본인부담, 비급여, 환자부담 총액, 비고), 합계(SUM 수식)가 포함됩니다.

## 기술 구성

단일 `index.html` 파일. pdf.js(PDF→이미지 변환), Puter.js(GPT-4o·Mistral OCR)·OCR.space·Tesseract.js(무료 AI 인식, 자동 전환), ExcelJS(색 있는 엑셀 생성)를 사용합니다.
