# 진료비 내역서 변환기 (Claude AI)

병원 진료비 영수증·내역서 **사진(JPG/PNG) 또는 PDF**를 업로드하면 **Claude AI**가 내용을 분석하여 실손의료보험 청구용 **엑셀 파일(.xlsx)** 로 변환해주는 웹 프로그램입니다. 양식이 병원마다 달라도 AI가 표 구조를 이해해 항목을 자동 매핑하므로, 진료일별 금액까지 정확하게 인식합니다.

**사용하기:** https://ds5tar.github.io/medical-bill-converter/

## 사용 방법

1. [console.anthropic.com](https://console.anthropic.com/) → API Keys에서 Claude API 키를 발급받습니다.
2. 앱 상단에 API 키를 입력합니다. (키는 브라우저에서 Anthropic API로만 전송되며, 저장 시 localStorage에만 보관됩니다)
3. 진료비 영수증 사진 또는 PDF를 업로드합니다. (여러 개 동시 선택 가능)
4. "AI 인식 시작"을 누르고, 결과를 표에서 확인·수정한 뒤 "엑셀 다운로드"를 누릅니다.

## 알아둘 점

- 인식을 위해 이미지가 Anthropic(Claude) 서버로 전송됩니다.
- API 사용료는 입력한 키의 소유자에게 부과됩니다(30페이지 기준 약 $0.2~0.3, 모델에 따라 다름).
- 사용 모델: `claude-sonnet-5` (비전).

## 출력 엑셀 양식

시트 "진료비내역"에 환자·의료기관 정보와 진료일별 내역(진료일자, 진료비 총액, 본인부담금, 공단부담금, 전액본인부담, 비급여, 환자부담 총액, 비고), 합계(SUM 수식)가 예제 파일과 동일한 색 서식(헤더 진한 파랑, 합계 연파랑, 라벨 회색, 테두리)으로 포함됩니다.

## 기술 구성

단일 `index.html` 파일. pdf.js(PDF→이미지 변환), Claude Messages API(비전 인식, 사용자 키), ExcelJS(색 있는 엑셀 생성)를 사용하며 브라우저에서 동작합니다.
