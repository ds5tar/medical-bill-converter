# 진료비 내역서 변환기

병원 진료비 영수증·내역서 **사진(JPG/PNG) 또는 PDF**를 업로드하면 **무료 OCR**로 내용을 인식하여 실손의료보험 청구용 **엑셀 파일(.xlsx)** 로 변환해주는 웹 프로그램입니다. 가입·API 키·비용이 전혀 필요 없습니다.

## 무료 OCR (한도 걱정 없음)

1. **OCR.space** — 무키 한국어 OCR (기본)
2. **Tesseract** — 브라우저 내장, 오프라인, 항상 동작 (자동 전환 안전망)

기본은 OCR.space를 쓰고, 한도·오류가 나면 오프라인 Tesseract로 자동 전환되어 인식이 멈추지 않습니다.

> 참고: 순수 OCR 방식이라 환자명·병원명·사업자번호·날짜 등은 잘 인식하지만, 표 안의 세부 금액(본인부담·공단부담 등)은 부정확할 수 있습니다. 결과 표에서 금액을 확인·수정한 뒤 다운로드하세요.

**사용하기:** https://ds5tar.github.io/medical-bill-converter/

## 사용 방법

1. 진료비 영수증 사진 또는 PDF를 업로드합니다. (여러 개 동시 선택 가능)
2. "AI 인식 시작"을 누릅니다. 페이지당 10~30초 정도 걸립니다.
3. 인식 결과를 표에서 확인·수정합니다.
4. "엑셀 다운로드"를 누르면 완성된 파일이 저장됩니다.

## 알아둘 점

- OCR.space 인식 시 이미지가 OCR.space 서버로 전송됩니다. Tesseract 단계는 전송 없이 내 브라우저에서 처리됩니다.
- 규칙 기반 파서가 OCR 텍스트에서 항목을 채우며, 특히 금액은 표에서 확인·수정이 필요합니다.

## 출력 엑셀 양식

시트 "진료비내역"에 환자·의료기관 정보와 진료일별 내역(진료일자, 진료비 총액, 본인부담금, 공단부담금, 전액본인부담, 비급여, 환자부담 총액, 비고), 합계(SUM 수식)가 포함됩니다.

## 기술 구성

단일 `index.html` 파일. pdf.js(PDF→이미지 변환), OCR.space·Tesseract.js(무료 OCR, 자동 전환), ExcelJS(색 있는 엑셀 생성)를 사용합니다.
