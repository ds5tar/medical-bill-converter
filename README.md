# 진료비 내역서 변환기

병원 진료비 영수증·내역서 **사진(JPG/PNG) 또는 PDF**를 업로드하면 **무료 AI**(Puter.js 기반 GPT-4o 비전)가 내용을 인식하여 실손의료보험 청구용 **엑셀 파일(.xlsx)** 로 변환해주는 웹 프로그램입니다. 가입·API 키·비용이 전혀 필요 없습니다. 최초 1회 Puter 서비스 이용 동의 창(무료)이 표시됩니다.

**사용하기:** https://ds5tar.github.io/medical-bill-converter/

## 사용 방법

1. 진료비 영수증 사진 또는 PDF를 업로드합니다. (여러 개 동시 선택 가능)
2. "AI 인식 시작"을 누릅니다. 페이지당 10~30초 정도 걸립니다.
3. 인식 결과를 표에서 확인·수정합니다.
4. "엑셀 다운로드"를 누르면 완성된 파일이 저장됩니다.

## 알아둘 점

- 인식을 위해 이미지가 Puter AI 서버로 전송됩니다.
- 최초 1회 Puter 이용 동의 창이 뜨며(무료), 이후에는 뜨지 않습니다.
- 병원마다 양식이 달라도 AI가 항목을 자동 매핑하며, AI 응답이 불완전하면 규칙 기반 파서가 보조합니다.

## 출력 엑셀 양식

시트 "진료비내역"에 환자·의료기관 정보와 진료일별 내역(진료일자, 진료비 총액, 본인부담금, 공단부담금, 전액본인부담, 비급여, 환자부담 총액, 비고), 합계(SUM 수식)가 포함됩니다.

## 기술 구성

단일 `index.html` 파일. pdf.js(PDF→이미지 변환), Puter.js(무료 GPT-4o 비전 인식), SheetJS(엑셀 생성)를 사용합니다.
