# 진료비 내역서 변환기

병원 진료비 영수증·내역서 **사진(JPG/PNG) 또는 PDF**를 업로드하면 브라우저 내장 **무료 OCR(Tesseract)** 이 내용을 인식하여 실손의료보험 청구용 **엑셀 파일(.xlsx)** 로 변환해주는 웹 프로그램입니다. 서버·API 키·비용이 전혀 필요 없으며, 모든 처리가 사용자 컴퓨터 안에서만 이루어져 개인정보가 외부로 전송되지 않습니다.

## 사용 방법

1. 페이지를 열고 진료비 영수증 사진 또는 PDF를 업로드합니다. (여러 개 동시 선택 가능)
2. "인식 시작"을 누릅니다. 처음 실행 시 한국어 인식 데이터를 내려받느라 수십 초 걸릴 수 있습니다.
3. 인식 결과를 표에서 확인·수정합니다. (무료 OCR 특성상 흐린 스캔본은 오인식이 있을 수 있음)
4. "엑셀 다운로드"를 누르면 완성된 파일이 저장됩니다.

## 출력 엑셀 양식

시트 "진료비내역"에 환자·의료기관 정보와 진료일별 내역(진료일자, 진료비 총액, 본인부담금, 공단부담금, 전액본인부담, 비급여, 환자부담 총액, 비고), 합계(SUM 수식)가 포함됩니다.

## 기술 구성

단일 `index.html` 파일. pdf.js(PDF→이미지 변환), Tesseract.js(무료 한국어 OCR), SheetJS(엑셀 생성)를 CDN으로 사용하며 모든 처리가 브라우저에서 이루어집니다.

## GitHub Pages 배포

저장소에 `index.html`을 올린 뒤 Settings → Pages → Branch를 `main`으로 설정하면 `https://<계정명>.github.io/<저장소명>/` 에서 누구나 사용할 수 있습니다.
